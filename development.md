# Develoment

-   Install `poetry`
    
    ```bash
      pip install poetry
    ```

-   Clone this repo
    
    ```bash
    git clone git@github.com:rtcamp/frappe-local-setup.git
    ```

-   Install all the python dependencies using `poetry`
    
    ```bash
    poetry install
    ```

-   Now develop as much as you want.

-   How to run `fm-cli`
    
    ```bash
    poetry shell
    fm
    ```
